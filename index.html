notes
  • check out the console to see the log of each button you click
  • chaining operators calculates last values first, further chaining reassigns operator
  • if you manage to break the calculator, please email me at harith.akma@gmail.com to tell me how you did it! I'll want to fix it
  • there is keyboard support, try it out!
If using keyboard:
  • use backspace to delete
  • type "*", "x", or "X" to use multiply
  • type "n" or "N" to toggle negative
  • "=" or "Enter" will activate equals-to button
  • the spacebar key clears the calculator
