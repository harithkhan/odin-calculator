  • There is keyboard support, try it out!
If using keyboard:
  • use backspace to delete
  • type "*", "x", or "X" to use multiply
  • type "n" or "N" to toggle negative
  • "=" or "Enter" will activate equals-to button
  • the spacebar key clears the calculator
